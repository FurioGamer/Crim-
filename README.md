# 🔍 CRIM'

### *Plongez au cœur de l'enquête.*

------------------------------------------------------------------------

## 🚨 Présentation

### 🎯 Pitch

**CRIM'** est un jeu vidéo d'enquête immersive développé en Python, dans
lequel vous incarnez un inspecteur de la Police Judiciaire. Votre
mission : résoudre des affaires criminelles complexes à partir d'indices
bruts, de rapports officiels et d'interrogatoires.

Aucune solution toute faite. Aucune piste imposée.\
Seulement vos capacités de déduction.

------------------------------------------------------------------------

### 🧠 Genèse

Le projet **CRIM'** est né d'un constat simple : la majorité des jeux
d'enquête sont linéaires et guident trop le joueur. Ici, l'objectif est
de recréer une expérience réaliste, où chaque décision compte et où
l'erreur fait partie du processus.

------------------------------------------------------------------------

### ⚖️ Objectifs

-   Offrir une liberté totale d'analyse au joueur\
-   Simuler des enquêtes réalistes basées sur des preuves concrètes\
-   Développer un système où chaque indice a du sens\
-   Créer une immersion proche du travail réel d'un inspecteur

------------------------------------------------------------------------

## ⚙️ Installation

### 1. Cloner le projet

``` bash
git clone https://github.com/FurioGamer/Crim-
cd crim
```

### 2. Créer un environnement virtuel

``` bash
python -m venv venv
```

### 3. Activer l'environnement

**Windows :**

``` bash
venv\Scripts\activate
```

**Mac / Linux :**

``` bash
source venv/bin/activate
```

### 4. Installer les dépendances

``` bash
pip install -r requirements.txt
```

### 5. Lancer le jeu

``` bash
python main.py
```
